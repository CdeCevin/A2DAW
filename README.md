## Contexto del trabajo
El proyecto se centra en la gestión de una clínica veterinaria, centralizando información de pacientes, dueños y personal. De esta manera, resuelve problemas como la pérdida de historiales clínicos, descoordinación de agendas y falta de trazabilidad de diagnósticos 
a través de un panel unificado con historiales médicos completos por mascota, centralización de información y variabilidad por roles.
